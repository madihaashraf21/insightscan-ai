# InsightScan - AI Document Analyzer
InsightScan is an AI-powered web application that analyzes text documents. 
It generates summaries and extracts key information such as names, locations, and dates using Natural Language Processing (NLP).

##  Features
- Text Summarization using Transformers
- Named Entity Recognition (NER)
- Flask-based Web Application
- Clean User Interface (HTML/CSS)

- ##  Tech Stack
- Python
- Flask
- Hugging Face Transformers
- spaCy
- HTML/CSS

##  How to Run
1. Install dependencies:
2. pip install -r requirements.txt
 python app.py
 3. Open in browser:
http://127.0.0.1:5000
